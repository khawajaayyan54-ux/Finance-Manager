# Finance-Manager
A Python-based CLI finance tracker that lets users record income and expenses securely using JSON files. It validates input, handles errors gracefully, calculates totals, and displays formatted summaries with balance in hand. Perfect for personal budgeting and learning automation.

## Features
1. Add income with reason and amount
2. Add expenses with reason and amount
3. Validate input to prevent crashes
4. Prevent negative numbers
5. Show summary with total income, total expenses, and balance
6. Formatted output with commas for readability

## Technologies Used
1. Python 3.x
2. JSON for data storage
3. OS module for file handling
